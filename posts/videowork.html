开卷 #

循环一： #

男主：我睡着啦？1：44，我怎么会在考试的时候睡着啊。哦原来剩下的题我都不会做了（背景音）谁考试的时候不关手机啊（爆炸）

循环二 #

男主：助教 我要交卷

助教：不准离开

男主：为什么

助教：考试结束前30分钟内不能交卷

男主：你让我走（爆炸）

循环三 #

男主：不行 根本没有办法强行离开考场。有人作弊，他抄我卷子

同学1：谁抄你卷子了

男主：他抄我卷子，我要去教务处举报他。

助教：去吧（爆炸）

张老师：同学你好，我是张老师。在你离开45号考场之后，考场发生了爆炸，我们认为考场有炸弹，想向你了解一些情况。

男主：我……陷入了循环。

张老师：咳……

张老师：杜老师你看到了吗？这小子一直跟我扯什么循环。

杜老师：准备药检

循环四 #

男主：所以按照老师说的，考场真的有炸弹。大家快跑，考场有炸弹。

同学1：卷疯了吧这个人

卢同学：想把我们都吓跑然后自己拿高分是吧

男主：为什么不信我啊（爆炸）

循环五 #

男主：炸弹提前爆炸了，也就是说引爆的人就在现场，我得直接去找张老师。有人作弊，他抄我卷子。

同学1：谁抄你卷子了

男主：你一定要相信我，我真的进入了循环。

张老师：你说你已经进入循环5次了，5次了为什么你还是只考了18分。

男主：张老师，都爆炸了，谁会用循环去做卷子啊

张老师：也对啊。不对，你怎么知道我姓张，见面的时候我又没跟你自我介绍，我又不认识你，你是怎么知道我姓张的。

男主：你给我上了半年课了，你怎么会不认识我啊。

杜老师：准备药检。

循环六 #

男主：我还是自己找炸弹吧，我不想再准备药检了。诶对了，我记得这个人一直护着自己的包，里面不会是炸弹吧，我得验他。给我看看啊

卢同学：你要干什么

男主：这是考试的开卷材料

卢同学：我辛辛苦苦准备了那么多的材料，考之前谁问我要我都说没准备，这样我就暴露了嘛

循环七 #

男主：既然如此我就一个一个的验。给我看看啊

同学1：不许动

循环八 #

男主：给我看看啊

循环九 #

男主：给我看看啊

同学2：我的任务完成啦（爆炸）

循环十 #

张老师：45号考场确实发生了一起作弊案件，当时作弊的人叫王猛猛，他被直接开除了学籍。

男主：卢·卷之使徒·高英终结者·被绩点选中的人·笛，我需要你的帮助。王猛猛作弊了被开除也是没有办法的事情啊

同学2：我哥没有作弊

男主：你干什么

助教：你们根本就不懂（爆炸）

循环十一 #

张老师：你们的助教5年前是王猛猛的室友

男主：如果有一天你收到一条短信，说有个考场会在20分钟后爆炸，您会相信他，然后立刻过来吗》

张老师：我会毫不犹豫地把发短信的人抓过来，考试的时候怎么能用手机呢

结局 #

男主：我一定要救下所有人。
